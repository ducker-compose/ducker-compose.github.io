1. Скрипт

профиль терминала
powershell, не cmd или bash (выбирается при наведении на + над окном терминала в vscode)

если у тебя не создан проект или тебе нужен распакованный архив

iex (irm https://clck.ru/3SeWji) # качаешь и распаковываешь архив в папку /front/
cd front                          # переходишь в папку
npm i                             # устанавливаешь зависимости
npm run dev                       # запускаешь проект

если ты уже в /front/ (папке, которую выбрал, когда открывал vscode, должна называться front и никак иначе, чтобы скрипт отработал)

cd ..                             # выходишь в папку, в которой находится твой проект
iex (irm https://clck.ru/3SeWji) # качаешь и распаковываешь архив в папку /front/
cd front                          # переходишь обратно
npm i                             # устанавливаешь зависимости
npm run dev                       # запускаешь проект

3. GitHub + git

если у тебя не создан проект

git clone https://github.com/ducker-compose/front.git # качаешь проект в папку /front/
cd front                                               # переходишь в папку
rmdir .git                                             # удаляешь информацию о репозитории чтобы не спалили
npm i                                                  # устанавливаешь зависимости
npm run dev                                            # запускаешь проект
